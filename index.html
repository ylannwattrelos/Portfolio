IUT'errain : Le Labyrinthe en Java

IUT'errain est un jeu de labyrinthe développé dans le cadre d'un projet universitaire. L'objectif était de mettre en pratique les connaissances acquises en programmation orientée objet (POO) et en développement d'interfaces graphiques, en utilisant le principe du MVC (Modèle, Vue, Contrôleur).

Technologies utilisées :

  • Java : Langage de programmation principal.
  • JavaFX : Framework pour la création de l'interface graphique.
  • FXML : Langage de balisage pour définir la structure de l'interface utilisateur, séparant la vue du contrôleur.

Le jeu propose différentes générations de labyrinthes et une navigation utilisateur fluide.

Check.exe : Échecs en Terminal

Check.exe est un jeu d'échecs entièrement développé en Java, conçu pour s'exécuter dans un terminal. Ce projet nous a surtout permis de découvrir et d'appliquer la méthodologie Agile.

Technologies utilisées :

  • Java : Utilisation intensive de la POO pour modéliser les pièces et leurs déplacements.
  • Bash/Scripts : Automatisation de la compilation et de l'exécution via des scripts shell.
  • Logique de jeu : Implémentation des règles spéciales (Roque, Prise en passant, Promotion).

Le jeu propose une interface en caractères ASCII pour une expérience rétro et légère.
